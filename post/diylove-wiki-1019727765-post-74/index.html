材料

  1. 建议来的时候带电脑，方便写作
  2. 来之前注册好GitHub.com账号

过程

  1. 实践其中一条指南（适合现场的成稿指南：https://github.com/diylove/wiki/projects/16）
  2. 介绍：diy💔wiki是什么？（github.com/diylove/wiki）
  3. 自我介绍 + 看完介绍后你有什么问题？
  4. GitHub 实操演示和Q&A 💻
    • 看 CODE 里面的文档、
    • 看 ISSUES 里面的指南和标签：diy一次旷课、diy一次行为艺术、diy一次苏格拉底式对话 ……
    • 创建指南
  5. 指南阅读和写作 🔇 20min
  6. 写作作品挨个儿展示和讨论，有的可以现场实践！
